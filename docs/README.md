## ğŸš€ Getting Started

### Development Setup

1. **Clone the repository**

   ```bash
   git clone [repository-url]
   cd WRH-lab-website
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start development server**

   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:3000
   ```

### Production Deployment

1. **Build the project**

   ```bash
   npm run build
   ```

2. **Test production build**

   ```bash
   npm run start
   ```
